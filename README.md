# Netflix EDA Project

This project performs Exploratory Data Analysis (EDA) on Netflix Movies and TV Shows dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Duplicate Handling
- Data Visualization
- Trend Analysis

## Key Insights
- Movies are more common than TV Shows
- USA produces highest Netflix content
- Netflix content releases increased rapidly in recent years
